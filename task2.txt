Difference between document & window object

Document
1.It represents any HTML document.
2.It is the object of window property.
3.The document is part of BOM and DOM.
4.All the tags, elements with attributes in HTML are part of the document.
5.It is loaded inside the window.
6.syntax:
     document.propertyname;

Window
1.It represents a browser window.
2.It is the object of the browser.
3.The window is part of BOM.
4.Global objects, functions, and variables of JavaScript are members of the window object.
5.It is the very first object that is loaded in the browser.
6.syntax:
    window.propertyname;
